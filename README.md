# Calculadora
# Totalmente-em-C
Uma calculadora baseada na linguagem C, que vou aprimorando, adicionando funções e evoluindo a cada "push".

Objectivos:
- Receber os numeros no terminal e devolver o resultado.
- Evoluir de acordo com os conhecimentos em C (implementar novas funções de calculos).

Implementações:
1. Operações aritméticas:
- Adição.
- Subtração.
- Multiplicação.
- Divisão.
2. Implementação de funções para cada função.
3. Implementação de calculo de percentagem.
